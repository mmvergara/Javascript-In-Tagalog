# trimEnd

## Pagpapaliwanag
trimStart method ay aalisin niyalang yung mga spaces sa hulihan ng string.
>Spaces like '\n' and '\t' included sila

```javascript
const x = "    hello    ";
console.log( x.length ) // 13

//Return Value
console.log( x.trimStart() ) //      hello     
console.log( x.trimStart().length ) // 9
```
