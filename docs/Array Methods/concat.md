# concat
## Pagpapaliwanag
Pagsama ang one or more array. pwede kahit ilang (array) arguments.

```javascript
const x = [1,2,3]
const y = [4,5,6]
const z = [7,8,9]

const a = x.concat(y,z)
console.log(a) // [1, 2, 3, 4, 5, 6, 7, 8, 9]
```
