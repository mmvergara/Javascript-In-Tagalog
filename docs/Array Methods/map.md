# map

## Pagpapaliwanag
Mag a-apply ng function sa each element ng array

```javascript
const x = [ 1, 2, 3, 4, 5 ]

const x10 = x.map( ( eachElement ) => { return eachElement*10 } )
console.log(x10) // [ 10, 20, 30, 40, 50 ]
```
