# includes 
## Pagpapaliwanag
Check if yung binigay mong element ay present sa array. <br/>
`Boolean` return value

```javascript
const array = [1, 2, 3, 4, 5]

console.log( array.includes(1) ) // true
console.log( array.includes(5) ) // true
console.log( array.includes(6) ) // false
```
