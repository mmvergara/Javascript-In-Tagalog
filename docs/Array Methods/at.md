# at

## Pagpapaliwanag

Kukunin niyalang ang value depende sa binigay mong `index` as a parameter

```javascript
const x = [0, 1, 2, 3, 4];

console.log( x.at(0) ); // 0
console.log( x.at(1) ); // 1
console.log( x.at(2) ); // 2
```
