# reverse

## Pagpapaliwanag
I re-reverse lang ang array

```javascript
const x = [ 1, 2, 3, 4, 5 ]
const returnValue = x.reverse()

console.log(x) // [ 5, 4, 3, 2, 1 ]
console.log(returnValue) // [ 5, 4, 3, 2, 1 ]
```
